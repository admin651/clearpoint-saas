Auth Quickstart

Use the following endpoints to create your first user and log in.

POST /auth/register
{
  "email": "you@example.com",
  "name": "Your Name",
  "password": "choose_a_strong_password"
}

POST /auth/token (form-data)
  username: you@example.com
  password: choose_a_strong_password

GET /auth/me (with Bearer token)

Then use your token to upload and process jobs at /jobs/upload and list them at /jobs/list.
